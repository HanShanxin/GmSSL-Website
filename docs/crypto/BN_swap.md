# NAME

BN\_swap - exchange BIGNUMs

# SYNOPSIS

    #include <openssl/bn.h>

    void BN_swap(BIGNUM *a, BIGNUM *b);

# DESCRIPTION

BN\_swap() exchanges the values of _a_ and _b_.

[bn(3)](http://man.he.net/man3/bn)
